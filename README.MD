# АртМинд — Интеллектуальная система психоэмоционального анализа детских рисунков

Веб-приложение для педагогов и психологов. Специалист загружает фотографию детского рисунка, указывает возраст ребёнка и контекст — система выдаёт психологический отчёт: доминирующие эмоции, риск-факторы, рекомендации.

## Архитектура

**Гибридная модель:** rule-based Computer Vision (8 модулей OpenCV) + семантический анализ Vision LLM с настраиваемой агрегацией весов (65/35).

```
Рисунок → OpenCV (8 модулей)  ──┐
                                 ├── Гибридная агрегация → Отчёт
Рисунок → Groq Vision (LLaMA-4) ┘
```

### OpenCV-анализатор (8 модулей)

| # | Модуль | Методика | Вес |
|---|--------|----------|-----|
| 1 | Цвет → эмоции | Тест Люшера | 0.25 |
| 2 | Зональный анализ | Методика Маховер | 0.20 |
| 3 | Анализ линий | Нажим, характер, хаотичность | 0.15 |
| 4 | Сигнатуры радости | Детектор позитивных паттернов | 0.12 |
| 5 | Детектор объектов | Контурный анализ | 0.13 |
| 6 | Haar-каскады | Детектор лиц и улыбок | 0.08 |
| 7 | LBP текстура | Анализ текстуры штриха | 0.04 |
| 8 | FFT спектр | Частотный анализ хаотичности | 0.03 |

### Психологическая база

- **Тест Люшера** — цвет → эмоции (10 цветовых групп)
- **Методика Маховер** — зональный анализ листа (верх/низ/лево/право/центр)
- **Арт-терапия Копытина** — общие принципы интерпретации

### 5 эмоций × 3 состояния

Эмоции: радость, грусть, тревога, агрессия, спокойствие.
Итоговые состояния: `норма` / `требует_внимания` / `требует_консультации`.

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/analyze` | Анализ рисунка (параметры: file, mode, age, context, save) |
| GET | `/health` | Статус системы (OpenCV, Groq, БД) |
| GET | `/groq/status` | Статус Groq API |
| GET | `/history` | Список анализов (limit, offset) |
| GET | `/history/{id}` | Детали анализа |
| GET | `/history/{id}/image` | Изображение рисунка |
| DELETE | `/history/{id}` | Удалить анализ |

### Режимы анализа

- `auto` — Groq hybrid с fallback на OpenCV
- `opencv` — только OpenCV (8 модулей)
- `groq` — только Groq Vision
- `hybrid` — Groq 65% + OpenCV 35%

## Запуск

### 1. Подготовка

```bash
# Создать БД PostgreSQL
createdb artmind

# Настроить переменные окружения
cd core
cp .env.example .env
# Отредактировать .env — указать DATABASE_URL и GROQ_API_KEY
```

### 2. Бэкенд

```bash
cd core
py -3.11 -m pip install -r requirements.txt
py -3.11 -m uvicorn server:app --reload --port 8000
```

### 3. Фронтенд

```bash
cd interface
npm install
npm run dev
```

### 4. Открыть

```
http://localhost:5173
```
